# Welcome

*Mega Scripts* are several scripts. Each script is designed to automate working with *Megatools - command line client for Mega.nz*.

These scripts are written with automation in mind. Giving a more streamed lined way to create other scripts that backup files to *Mega.nz* account(s) with logging and email options.

See [Online Help file](https://amourspirit.github.io/mega_scripts/)
