Script Name: mega_dir_file_exist.sh

Description

Script check to see if a directory or file exist on Mega.nz.


Parameters

Param 1

Optional: The path to test if it exist
Eg: "/MyPath/myfile"

Param 2

Optional: The configuration file to use that contain the credientials for the Mega.nz account you want to access.

Possible Codes returned from Script

Code

Defination

0

Directory not found

1

Directory is found and is Root

2

Directory found

3

File Found

102

megals not found. Megtools requires installing

111

Optional argument two was passed in but the config can not be foud or we do not have read permissions


Remarks

Never pass in the directory Root to mega_dir_file_exist.sh as the frist part of the path.

For Param 1 "/MyPath/myfile" is fine; however, "/Root/MyPath/myfile" will fail.


If Param 2 is omitted then you must have .megarc set in the current users home directory.


The mega_dir_file_exist.sh does not have any logging or output to the terminal. Only the Exit code is set.


Example

Script example to check and see if a file or folder exist on default Mega.nz account on the command line.


# /bin/bash /usr/local/bin/mega_dir_file_exist.sh '/MyPath/myfile'; echo $?


If the file exsit above then 3 will echo to the command prompt if file does not exsit then 0 will echo.

Example

Script example to check and see if we have access to Mega.nz


# /bin/bash /usr/local/bin/mega_dir_file_exist.sh; echo $?


or to check a non-default account


# /bin/bash /usr/local/bin/mega_dir_file_exist.sh "" ~/.megarc_other; echo $?


If the file exsit above then 1 will echo to the command prompt if file does not exsit then 0 will echo.
